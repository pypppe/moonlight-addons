# Moonlight Addons

Welcome to **Moonlight Addons**! Follow the instructions below to get started.

---

## Windows Installation

1. **Install required Python packages**  
   Open **Command Prompt** and run:

   `pip install psutil pillow wmi`

2. **Extract the ZIP file**  
   Unzip the downloaded `Moonlight_Addons` archive to a folder of your choice.

3. **Navigate to the project directory**  
   In Command Prompt, change to the folder where you extracted the files. For example:

   cd C:\Users\USERNAME\Downloads\Moonlight_Addons\moonlight_addons

4. **Run the program**  

   `python moonlightaddons.py`

   You should see a message like:

   **Moonlight Addons have been installed.**

   ---
## Linux INstallation

# 1. Install required Python packages
pip3 install psutil pillow wmi
# Note: 'wmi' is Windows-specific. On Linux, it can be skipped unless needed.

# 2. Extract the ZIP file
unzip Moonlight_Addons.zip -d ~/Moonlight_Addons

# 3. Navigate to the project directory
cd ~/Moonlight_Addons/moonlight_addons

# 4. Run the program
python3 moonlightaddons.py

# You should see:
# Moonlight Addons have been installed.
